INSTRUCTIONS
------------
Put all files in one directory.

Look at main.c which is a minimalist example of how you can create an
image and save it in .gif.
You have to modify this program to decide what to do with every pixel (x,y)

You must compile doing 
cc main.c image.c -o programa (C)

gcc  main.c image.c -o programa (C++)

This creates an executable file called programa.
Run it by
./programa
Its execution creates a file called test.gif (you can change 
this name from inside main.cc). 

The color palette that it uses (blue.map) is also indicated from the inside of main.c.
The files *.map are different color palettes, which assign an RGB color (3 numbers)
to every number from 0 to 255. You can create your own palette files, but some examples are in the directory.

You may modify the program as you see fit.


INSTRUCTIONS FOR COMPLEX
-------------------------
(using cc compiler for C, in linux)

If you want to use complex numbers you need to add
#include<complex.h>
#include<math.h>

to the main program. Then, you have three new types

float complex
double complex
long double complex

and also a predefined number I which is the complex number 0.0 + 1.0*I
You may define complex numbers like z = x + I*y
where x and y are floats or doubles or long doubles.
You also have several functions like 
cabs(), creal(), cimag(), ...

REMEMBER TO COMPILE USING -lm TO LINK TO THE MATH LIBRARY

cc -lm main.c image.c -o programa
